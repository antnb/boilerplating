# Layout Components

**Purpose:** Global layout components used across ALL pages.

## Included Components:
- `Navbar.tsx` - Main navigation bar (desktop + mobile)
- `Footer.tsx` - Footer with 4 columns (Brand, Explore, Company, Contact)
- `Logo.tsx` - BMJ brand logo
- `Breadcrumbs.tsx` - Navigation breadcrumbs
- `Section.tsx` - Standard section wrapper with max-width and padding
- `SectionHeading.tsx` - Reusable section title component

## Rules:
✅ **DO:** Put components used in root layout or shared across ALL pages
❌ **DON'T:** Put page-specific components here
